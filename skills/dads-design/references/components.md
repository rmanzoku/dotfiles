# DADS コンポーネント一覧（46種）

デジタル庁デザインシステム（DADS, β版）が提供するUIコンポーネント。出典: `https://design.digital.go.jp/dads/components/`（実DOMより全件取得）。

- 一覧ページにカテゴリ分けは無く、46個がフラットに並ぶ。
- 各コンポーネントのURL: `https://design.digital.go.jp/dads/components/<slug>/`。
- 実装（HTML/CSS/JS, React/Tailwind）はGitHub `digital-go-jp/design-system-example-components-html` / `-react`（いずれもMIT）と各Storybookを参照。本スキルは実装コードを同梱しない。
- 各コンポーネントページには達成基準バッジ（`JIS2016` = JIS X 8341-3:2016／WCAG 2.0 と同一内容、`WCAG2.2`）が表示される。
